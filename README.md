# SteamServ
 A tool for receiving steam server information in Discord.
 This is an updated version of a previous build of realSaddy's (which used Node.js).
### Env File
```
TOKEN=
MONGO_USERNAME=
MONGO_PASSWORD=
MONGO_HOST=
MONGO_DATABASE=
```

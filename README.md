# SteamServ
 A tool for receiving steam server information in Discord.
 You can use the official, hosted version here:
 
 [![Discord Bots](https://top.gg/api/widget/441389975951507486.svg)](https://top.gg/bot/441389975951507486)
### Env File
```
TOKEN=
MONGO_USERNAME=
MONGO_PASSWORD=
MONGO_HOST=
MONGO_DATABASE=
DBL_TOKEN=
DEBUG=
```
